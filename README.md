# Personal Portfolio – Frontend

This is the **frontend** for Akenna Kublal's personal portfolio website.  
It is built with **React + Vite**, styled with **Tailwind CSS**, and deployed on **Netlify** (or Vercel).

---

## ✨ Features
- Custom pages: Home, About, Speaking, Coaching, Contact
- Fully responsive design
- Contact form powered by [EmailJS](https://www.emailjs.com/)
- SEO-friendly with meta tags
- Integrated with Strapi CMS for dynamic content

---

## 🛠️ Tech Stack
- **React** (Vite)
- **Tailwind CSS** for styling
- **React Router** for navigation
- **EmailJS** for sending contact form submissions
- **Deployed on Netlify**
